Frequently Asked Questions

When downsizing, quality loss is minimal. When upsizing, some quality loss is expected since the browser has to interpolate new pixels. For best results, avoid scaling up more than 2x the original size.
Currently this tool processes one image at a time. For batch resizing, you can process each image sequentially — the settings are preserved between uploads.
There's no fixed limit. Processing happens in your browser, so it depends on your device's RAM and CPU. Images up to 20MB or 8000×8000px work well on most devices.
